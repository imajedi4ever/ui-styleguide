Welcome to the UI Style Guide!

This guide is intended to foster rapid development of application user interfaces while preserving design consistency across all company products.

A user interface style guide can serve as:

  • A tool for ensuring consistency across a product set
  • A way to get teams to work together
  • The repository for design guidelines and standards
  • A training aid for new members of development teams
The creation of a style guide can serve as a focal activity for an integrated design team involving documentation, development, usability, graphic design, marketing, and other groups.

This style guide was constructed using Bootstrap3 with HTML5 and CSS3

Bootstrap is a "sleek, intuitive and powerful front-end framework for faster and easier web development." Bootstrap implements global CSS settings, fundamental HTML elements styled and enhanced with extensible classes, and 12-column responsive grid system. It features dozens of components, styles and JavaScript plugins, with basic global display, typography and link styles. Use the Customizer to make Bootstrap your own by adjusting variables, components, JavaScript plugins and more.

Bootstrap3's responsive utilities enable faster mobile-friendly development on any device.

The 'Mobile-first' framework is built to work best in the latest desktop and mobile browsers, meaning older browsers might display differently styled, though fully functional, renderings of certain components.

HTML5 and CSS3 are VERY quickly becoming the new standards.

Modern browsers all support HTML5 and CSS3 allowing developers to use all the new and exciting features it offers in order to create more dynamic end results.
